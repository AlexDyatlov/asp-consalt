1С:Комбинат планового питания

  • 1С:ИТС
  • 1С Для образовательных учреждений
  • Бухгалтерский и налоговый учет
  • Общественное питание и гостиничный бизнес
  • Здравоохранение и медицина

Специализированное решение для учета питания в высших учебных заведениях, индустриальных комбинатах питания, обеспечивающих поставку питания многим организациям-потребителям.

Программа адресована:

  • заведующему производством столовой,
  • бухгалтеру-калькулятору,
  • технологу питания (диетсестре),
  • бухгалтеру материального учета (кладовщику),
  • бухгалтеру по взаиморасчетам с потребителями.

Решаемые задачи

  1. Планирование питания
    • разработка и оформление блюд и типовых меню,
    • расчет заказа продуктов поставщику,
    • предварительная калькуляция стоимости блюд в учетных ценах продуктов.
  2. Повседневная работа с меню
    • составление и оформление "Меню-требования" с расчетом стоимости и пищевой ценности, заменой продуктов,
    • печать "Меню" для воспитанников и родителей,
    • ведение "Бракеражного журнала готовой продукции".
  3. Учет продуктов
    • заказ продуктов поставщику и ввод прихода от поставщика,
    • автоматическое списание продуктов по калькуляции или ручное списание,
    • автоматическое ведение остатков,
    • инвентаризация,
    • партионный учет складских единиц измерения, сроков хранения и санитарных сертификатов.
  4. Итоговые документы за период (месяц)
    • "Ведомости анализа стоимости питания",
    • "Накопительной ведомости по расходу продуктов",
    • "Сводки по пищевой ценности рациона".

Основные функциональные возможности программы:

  • Ведение картотеки блюд с нормами закладки продуктов, описанием технологии приготовления, сведениями о пищевой ценности. Описание состава продуктов блюда включает следующие сведения: нетто, брутто, отход холодной обработки, потери при кулинарной обработке, масса продуктов после кулинарной обработки, структура технологических процессов;
  • Ведение типовых циклических меню по нескольким рационам;
  • Ведение номенклатуры продуктов. По каждому продукту ведутся нормы отхода при холодной обработке, сведения о пищевой ценности, технические условия;
  • Ведение набора характеристик пищевой ценности переменного состава;
  • Разработка рецептур блюд и типовых меню;
  • Расчет заказа продуктов поставщику, включая возможность выгрузки электронной заявки;
  • Учет продуктов на складах: приход, расход, перемещения товаров, остатки, инвентаризация;
  • Суммовой учет остатков товаров по местам хранения;
  • Настраиваемый расчет стоимости с учетом льготирования по НДС, расчет продажных цен с использованием различных ставок наценки по ценовым категориям и видам расчетов;
  • Разделение учета по видам движения средств (источникам финансирования);
  • Опциональный партионный учет продуктов, по партиям ведутся сведения о сроках хранения, санитарных сертификатах, конкретных единицах измерения (банках, батонах и т.д.);
  • Калькуляция: составление и расчет меню и наряда цеху, распечатка меню-требования и калькуляционных карточек, учет норм отхода холодной обработки, замен продуктов и блюд, пробы, составление меню на дополнение и возврат. Автоматизированное списание продуктов и расчет стоимости питания;
  • Учет численности питающихся по получателям и категориям;
  • Учет планируемой поставки и планируемого расхода продуктов на следующие дни, дозаказ недостающих продуктов;
  • Бракераж готовых блюд с регистрацией оценок и распечаткой вкладных листов бракеражного журнала;
  • Контроль фактического рациона по стоимости, пищевой ценности, натуральным нормам потребления;
  • Массовые операции: распечатка наборов накладных по нескольким документам; распечатка настраиваемого пакета документов по накладным и меню; массовая выписка накладных по заявке на товары; массовое создание накладных по калькуляции;
  • Обеспечение параллельной работы нескольких бухгалтеров-калькуляторов: остатки на складах; обмен сообщениями;
  • Конфигурация является полностью открытой, не содержит защищенных участков кода и не использует аппаратную защиту.

Дополнительные функциональные возможности:

  • Выходные формы, принятые для учета в медицинских организациях (по приказу Минздравсоцразвития РФ №330 от 5 августа 2003 г.);
  • Ведение календаря типового меню;
  • Сводное типовое меню по нескольким рационам;
  • Расчет характеристик пищевой ценности блюд по продуктам;
  • Расчет норм отхода холодной обработки на поступившую партию сырья с оформлением "Акта проработки";
  • Выгрузка документов о движении продуктов в систему бухгалтерского учета с использованием механизма конвертации данных;
  • Выгрузка данных меню во внешний файл для сдачи в централизованную бухгалтерию;
  • Возможность загрузки рецептур, типовых меню, продуктов и сведений пищевой ценности из внешних источников (из XML-файла в формате "Рецептурник");
  • Выгрузка/загрузка данных в формате XML;
  • Динамическое подключение внешних отчетов и обработок;
  • Начальная загрузка данных из "1С:Комбинат Питания 7.7" и "1С:Школьное Питание 7.7".
